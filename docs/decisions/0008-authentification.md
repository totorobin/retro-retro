# Gestion de l'authentification

## Contexte

La nécéssité de géré des utilisateur sous entend géré également des mots de passes. pour éviter d'avoir a géré la sécurité nécéssaire pour protéger les mots de passe la gestion de l'authentification peux etre géré par un tier.

## Décision

Dans un premier temps, l'authentification des utilisateurs de l'application sera géré par Google Sign-In.
A l'avenir d'autres tier pour l'authentification pourrais etre ajoutés

## Conséquences

Google sera utilisé pour connecter les utilisateur.
Les informations a retournés suite a l'authentification seront :
- un identifiant unique
- l'adresse mail
- le nom et le prénom
- une image de profil
