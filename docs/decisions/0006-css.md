# Librairie css

## Contexte

l'utilisation d'une librairie css ou une librairie de composant peux se révélé compliqué dans la durée. en effet une librairie peux etre abandonée par ses créateur ou changer de version majeure. sont utilisation bloquerai alors la possibilité de monter en version le framework ou d'autres libs.

## Décision

Aucune libraire de composant ou de css externe ne sera utilisé.
le css sera utilisé dans l'ensemble des composants VueJs avec des variables css pour les couleurs, bordures et polices.
ces variables css seront déclaré dans un fichier css externe au composants
