# Rétro Rétro
Application de rétrospective agile.

## Installation

```bash
pnpm install
```

## Configuration

Avant de démarrer l'application, vous devez configurer les variables d'environnement.
Consultez le guide : [Configuration de Google OAuth2](docs/guides/google-oauth-setup.md).

## Démarrage

```bash
pnpm dev
```
